# 🌈✨ Anastasia Salter — Lisa Frank Portfolio ✨🌈

A markdown-driven portfolio site converting the CV of **Anastasia Salter** (Professor of
English, Director of Graduate Programs, President of the Electronic Literature Organization)
into a gloriously over-the-top, **Lisa Frank–inspired** experience with *way* too much animation. 🦄

## 🦄 What's inside

| File | What it is |
|------|------------|
| [`index.html`](index.html) | The live, animated portfolio site 🌟 |
| [`PORTFOLIO.md`](PORTFOLIO.md) | The portfolio content in plain Markdown 📝 |
| [`assets/styles.css`](assets/styles.css) | All the rainbow gradients, glow, wobble & sparkle 💖 |
| [`assets/script.js`](assets/script.js) | Floating emoji sky, sparkle cursor trail, confetti, counters 🐬 |
| `2026_Salter_CV.docx` | The original source CV 📄 |

## ✨ The "too much animation" feature list

- 🌈 Animated rainbow gradient background that never stops shifting
- 🦄 Floating emoji sky (unicorns, dolphins, rainbows, hearts) drifting upward
- ✨ Sparkle cursor trail that follows your every move
- 🎉 Confetti burst on every click
- 📊 Animated counting stat cards
- 💫 Scroll-reveal animations on every section
- 🌟 Rainbow gradient text, wobble, bounce, pulse, spin & neon glow
- 🎠 Scrolling marquee of research areas
- 🚀 Spinning rainbow back-to-top button

> ♿ **Accessibility:** all motion respects `prefers-reduced-motion` — animations
> collapse to near-instant for visitors who opt out.

## 🚀 Run it

It's a static site — no build step. Just open `index.html`:

```bash
# from the repo root
python3 -m http.server 8000
# then visit http://localhost:8000
```

Or open `index.html` directly in any browser. 🌐
